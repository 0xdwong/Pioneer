# 登链翻译计划

[登链社区](https://learnblockchain.cn/)是一个区块链技术极客社区，登链翻译计划是为国内区块链爱好者引入优质的区块链技术文章。


## 翻译指南

查看 [Wiki-如何参与翻译](https://github.com/lbc-team/Pioneer/wiki)
